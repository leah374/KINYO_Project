# 一根高清线，普通电视3步变家庭KTV+影院

## 最终策划
- target_user：家里有普通电视、想低成本升级家庭娱乐的家庭用户，尤其是经常朋友聚会、周末想在家唱歌看电影、又嫌去KTV麻烦和花费高的人群。
- creative_angle：用‘普通电视一根线秒变家庭KTV+影院’做强转化切入，先击中电视闲置和聚会娱乐单一的痛点，再用现场接线、点歌界面、曲库和赠品快速建立购买理由，最后用聚会场景和福利促单完成ROI导向收口。
- core_pain：家里电视利用率低，朋友来家里时缺少有气氛的娱乐项目；去KTV不方便、成本高；担心家庭K歌设备安装麻烦、功能单一、不值这个钱。
- core_selling_point：只要一根线连接普通电视，就能把客厅变成家庭KTV和影院；曲库多且实时更新，操作简单，还送麦克风，特别适合朋友聚会和家庭娱乐。

## 四段式脚本

### Hook（0-3s）
- 作用：反差/悬念
- 画面：首帧直接做强反差：0.0-0.8s冷场客厅，普通电视亮着但没人看，朋友各自低头刷手机；0.8s音乐卡点切黑再爆亮，电视瞬间跳出K歌点歌界面，Alice举起麦克风看向镜头。大字报依次爆出‘电视吃灰？’→‘一根高清线’→‘3步变KTV’。字幕和画面跟鼓点同步跳出。
- 口播：你家电视还在吃灰？其实只要一根高清线，3步就能把客厅变成家庭KTV。
- 字幕：电视还在吃灰？
一根高清线
3步变家庭KTV
- 镜头/剪辑：BGM建议120-128BPM轻快派对感；第1拍出冷场画面，第4拍切‘一根高清线’，第8拍亮出K歌界面；加‘whoosh’转场音效和电视亮屏‘ding’音效，首秒冲击更强。

### Setup（3-8s）
- 作用：建立场景/痛点
- 画面：Alice把家庭K歌影视一体机放上茶几，旁边清楚摆出2支麦克风和遥控器；xiaojun看着电视摊手，形成‘聚会没节目’的轻尴尬。镜头快速扫到普通电视、设备、麦克风同框，强化‘家里有电视就能升级’。最后定格在Alice手拿高清线，准备现场连接。
- 口播：朋友来家里，光聊天太容易冷场；出去唱歌又麻烦、还花钱。家里有普通电视，接上这台，唱歌看电影一台搞定。
- 字幕：朋友来家容易冷场
不想跑KTV
有电视就能升级
K歌+看电影一台搞定
- 镜头/剪辑：这一段保持快节奏，不铺陈参数；产品、电视、2支麦克风必须同框一次，为后面福利点和数字锚点做铺垫。

### Twist（8-24s）
- 作用：转折/解决
- 画面：实拍证明为主。8-12s特写Alice手部操作，高清线一头插设备、一头插电视接口，屏幕角落同步浮字‘第1步 插线 / 第2步 连接 / 第3步 点歌开唱’；12-14s电视立刻亮起点歌界面，出现热门榜、经典老歌、排行、合唱等分类；14-18s XiaoMing拿遥控器演示语音找歌，界面快速跳出结果；18-21s Alice和xiaojun拿麦克风开唱，客厅气氛明显升温；21-24s一键切到电影海报墙，朋友坐回沙发继续看。全程穿插接口特写、亮屏特写、界面特写、聚会中景。
- 口播：安装真的简单：插上线、连上电视、点歌就能唱。连上立刻就是点歌界面，热门榜、经典老歌、合唱歌单这些常用分类都有，曲库丰富还会持续更新。想找歌，直接用遥控器点也行，说出来也行，操作很顺手。朋友来了先开唱，唱累了再一键切到电影模式，客厅一下就有KTV加影院的感觉。
- 字幕：第1步 插线
第2步 连接电视
第3步 点歌开唱
连上立刻出现点歌界面
曲库丰富 持续更新
操作简单 说歌名/点歌都方便
唱歌累了 一键切电影
- 镜头/剪辑：按高转化证据强化‘现场接线+立刻亮界面’；接线插入时加‘click’音效，亮屏时加‘ding’，语音搜歌时加轻提示音，切电影模式时做一键滑动转场；字幕卡点跟BGM主拍走，避免口播和画面脱节。

### CTA（24-30s）
- 作用：评论钩/转发/点击引导
- 画面：客厅进入热闹状态，Alice和xiaojun各拿1支麦克风对唱，随后切到产品+2支麦克风整齐陈列特写，福利字卡占画面中心：‘下单送麦克风’。最后Alice指向屏幕下方，同时旁边浮出互动文案：‘你更想在家唱歌还是看电影？’与‘转给家里有电视的朋友’。
- 口播：现在下单还送麦克风，家里有电视就能玩。想升级客厅娱乐的，先点进来看看。你更想在家唱歌还是看电影？评论区告诉我，也可以转给那个总说聚会没节目的人。
- 字幕：下单送麦克风
有电视就能用
想升级客厅娱乐 点进来看看
你更想唱歌还是看电影？
转给家里有电视的朋友
- 镜头/剪辑：CTA同时兼顾转化和互动：先打福利，再抛评论问题，最后补转发触发；福利字卡至少停留1秒，产品和赠品数量清楚可见。
