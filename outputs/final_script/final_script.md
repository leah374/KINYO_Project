# 别让电视只看新闻，一根线直接变家庭KTV

## 最终策划
- target_user：家里有普通电视、想低成本升级家庭娱乐的家庭用户，重点覆盖有家庭聚会需求、经常在家陪长辈、觉得去KTV麻烦又花钱的人群。
- creative_angle：用“普通电视一根线秒变家庭KTV+家庭影院”的低门槛改造感做转化，先打消安装复杂和价格高的顾虑，再用现场连接、点歌界面、语音操作、影视切换和赠送麦克风强化下单理由。
- core_pain：家里电视闲置利用率低；去KTV不方便、成本高；想唱歌看电影往往要买多台设备；长辈不会复杂操作；家庭聚会缺少能让全家都参与的娱乐方式。
- core_selling_point：只需一根线连接普通电视，就能实现K歌+观影+多娱乐功能一体；曲库多且实时更新；语音点歌简单适合长辈；下单赠送无线麦克风，适合家庭聚会和居家娱乐转化。

## 四段式脚本

### Hook（0-4s）
- 作用：反差/悬念
- 画面：首秒用强反差快切：0.3秒黑屏电视/新闻画面，0.3秒全家无聊坐沙发，下一帧立刻切到同一台电视上的KTV点歌界面和家里开唱热闹画面。最后定格主持人站在电视旁，手里举着主机和一根高清线。
- 口播：别再让家里电视只看新闻了。1根线、2步连上，普通电视直接变家庭KTV，还能看电影。你家电视现在最常拿来看什么？
- 字幕：电视别闲着｜1根线2步连上｜秒变家庭KTV
- 镜头/剪辑：前1秒必须完成‘无聊-开唱’反差；字幕用超大字分三拍出现：‘别闲着’‘1根线’‘变KTV’；配合鼓点卡切，提升停留。

### Setup（4-9s）
- 作用：建立场景/痛点
- 画面：长辈拿遥控器找节目、孩子坐一旁、夫妻表情无奈；快切外出KTV麻烦的联想字幕，再切回茶几上的主机、遥控器、赠送无线麦克风同框展示，画面整洁，突出设备不复杂。
- 口播：出去唱歌要约时间、要花钱，在家想热闹又怕设备麻烦。尤其爸妈想唱老歌、听戏曲，操作一复杂就不想用了。这个就是把K歌和观影合到一起的一体机，不用换电视，也不用配一堆设备。
- 字幕：去KTV麻烦又花钱｜长辈怕复杂｜不用换电视
- 镜头/剪辑：2-3个生活痛点镜头快速交代，BGM不停顿；产品套装要给清楚同框，顺手带出赠品，为后面CTA埋点。

### Twist（9-24s）
- 作用：转折/解决
- 画面：核心演示采用连贯三连证据：先拍接口特写，主持人拿一根高清线，一端插主机、一端插电视；屏幕立刻亮起进入点歌首页；镜头给到电视界面特写，连续切热门榜、经典老歌、儿歌/家庭歌单等分类。随后演示对遥控器语音点歌，屏幕即时检索歌曲，长辈接过麦克风准备唱。再切同一台电视从K歌页切到电影/电视剧页面，角落轻量提示戏曲、健身、直播、跳舞等全家娱乐内容。
- 口播：安装就这么简单：1根线，2步连上，开机就进点歌页。这个界面就跟KTV点歌台一样，热门榜、经典老歌、家庭歌单都能直接找。曲库丰富，持续更新，家里人想唱的歌更容易找到。不会打字也没关系，直接对着遥控器说歌名，长辈也能轻松点。想看电影、追剧，再一切换就行，不只是K歌机，是客厅里的家庭影音娱乐一体机。
- 字幕：插线即用｜开机直达点歌页｜像KTV一样好找歌｜语音点歌 长辈也会｜K歌+影视一体
- 镜头/剪辑：这里是转化核心，严格按鼓点做‘插线-屏亮-点歌页-语音识别-影视切换’五连卡点；同一台电视完成全部操作，强化真实感；屏幕内容尽量占满画面，减少空讲。

### CTA（24-30s）
- 作用：评论钩/转发/点击引导
- 画面：家庭欢唱和长辈拿麦互动的热闹画面，与产品主机+无线麦克风同框特写交替出现。最后定格产品、电视点歌页、赠品大字卡三者同屏，底部做按钮式引导字幕。
- 口播：现在下单就送无线麦克风，家里聚会能唱，陪长辈在家也能看、也能玩。活动进行中，想把家里电视升级成家庭娱乐中心的，点进来先领福利。评论区告诉我，你家电视最想先拿来唱歌还是看电影？
- 字幕：下单送无线麦｜家庭聚会/陪长辈都合适｜点击了解活动
- 镜头/剪辑：最后3秒必须放大福利和行动指令；加‘评论区告诉我’提升互动；可在末帧补‘先领福利再下单’增强转化确定性。
