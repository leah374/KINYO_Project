# 自定义 Logo 说明

## 如何更换侧边栏 Logo

### 步骤 1: 准备 Logo 图片

推荐规格：
- **文件格式**：PNG、JPG、JPEG 或 WebP
- **推荐尺寸**：宽度 400-800px（高度自动缩放）
- **推荐宽度**：200px（侧边栏显示宽度）
- **背景**：透明背景效果最好

### 步骤 2: 上传 Logo

将你的 Logo 图片重命名为以下任一名称，放入 `assets/` 文件夹：

```
assets/logo.png   ← 推荐（透明背景）
assets/logo.jpg
assets/logo.jpeg
assets/logo.webp
```

### 步骤 3: 重启应用

保存图片后，刷新浏览器页面或重启 Streamlit：
```powershell
# 按 Ctrl+C 停止
python run_streamlit.py
```

## 示例

**原图（建议尺寸 400x100px）**：
```
┌────────────────────────────┐
│    Your Logo Here          │  ← 宽度 400px，高度 100px
│    KINYO                  │
└────────────────────────────┘
```

**侧边栏显示效果（自动缩放）**：
```
┌──────────────────┐
│  Your Logo Here  │  ← 宽度 200px
│  KINYO          │
└──────────────────┘
```

## 尺寸建议

| 用途 | 推荐宽度 | 推荐格式 |
|------|---------|---------|
| Logo 图片 | 400-800px | PNG（透明背景）|
| 图标 | 200-400px | PNG |
| 横幅 | 800-1200px | PNG/JPG |

## 快速测试

1. 将你的图片保存为：`assets/logo.png`
2. 刷新浏览器页面
3. Logo 应该出现在侧边栏顶部

## 注意事项

- 如果没有自定义 Logo，会显示默认的 🎬 KINYO 文字
- 支持的格式：`.png`, `.jpg`, `.jpeg`, `.webp`
- 侧边栏会自动将图片缩放到 200px 宽度
- 建议使用透明背景的 PNG 以获得最佳效果
